# FileShare

最終更新: 2026-06-26 / バージョン: 1.0.0

WordPress 上でセキュアにファイルを共有するためのプラグインです。
アップロードしたファイルを「ダウンロード ID + パスワード」で保護し、安全に相手へ受け渡せます。

## 主な機能

- 📤 **ファイルアップロード** — ドラッグ&ドロップ対応。複数ファイルは自動的に ZIP へ圧縮。
- 🌐 **フロントからのアップロード** — 固定ページ不要。専用URL（例: `?fileshare_upload=1`）にアクセスし、専用 ID/パスワード（ダッシュボードで設定）でログインしてアップロード。WordPress ログイン不要。
- 🗂 **ファイル管理** — 保存ファイルの一覧表示・削除（管理画面）。
- 🔐 **安全なダウンロード** — ダウンロード ID + パスワードによる認証。
- 🔢 **ダウンロード回数制限** — ファイルごとに最大回数を設定（0 = 無制限）。
- 🧹 **自動クリーンアップ** — 期限切れ・回数超過ファイルを WP-Cron で自動削除。
- 🛡 **IP ブルートフォース対策** — 連続ログイン失敗で IP を一時ブロック。
- ⏱ **レート制限** — エンドポイント単位でアクセスを制限。
- 🌙 **ダークテーマ UI** — GitHub ライクな見やすいインターフェース。

## 動作環境

- WordPress 6.2 以上
- PHP 8.1 以上
- PHP `sqlite3` 拡張（ログ機能で使用）
- PHP `zip` 拡張（複数ファイルの圧縮で使用）

## インストール

1. `fileshare` フォルダを `wp-content/plugins/` に配置します。
2. 管理画面「プラグイン」で **FileShare** を有効化します。
   （PHP/WordPress バージョンや必要な拡張が不足している場合は有効化が中断されます。）

## 使い方の概要

1. 管理メニュー **FileShare** からファイルをアップロードします。
2. 表示される「ダウンロード ID」「パスワード」「ダウンロードページ URL」を相手に共有します。
   ダウンロードページは専用URL（`?fileshare_download=1`）で提供され、固定ページの設置は不要です。
   アップロード結果の URL（例: `?fileshare_download=1&fsid=XXXX`）をそのまま共有できます。

### フロントからアップロードさせたい場合（社内向け）

1. 管理メニュー **FileShare → フロント設定** で専用 ID/パスワードを設定し、有効化します。
2. 同じ画面に表示される **専用アップロードURL**（例: `https://example.com/?fileshare_upload=1`）を担当者に共有します。固定ページやショートコードは不要です。
3. 担当者はそのURLにアクセスして ID/パスワードでログインし、アップロードできます（ログイン状態は 1 時間保持）。

詳細な手順は [USE-Plugin.md](./USE-Plugin.md) を参照してください。

## セキュリティについて

- 実ファイルは公開ディレクトリ外参照を想定したランダム名で保存し、`.htaccess` で直接アクセスを遮断します。
- パスワードは bcrypt でハッシュ化して保存します。
- 認証成功時のみ短命の署名付きトークンを発行してファイルを配信します。

> ⚠️ **Nginx をお使いの場合**、`.htaccess` は効きません。
> `location ~ /uploads/fileshare-files/ { deny all; }` 相当の設定を別途追加してください。
